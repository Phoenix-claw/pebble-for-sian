# Pebble for Sian

Pebble is a small soft pixel-storybook Windows desktop companion, originally
created as a gift for Sian. Everything else is better discovered by spending
time with her.

## Download

Use the [latest release](https://github.com/Phoenix-claw/pebble-for-sian/releases/latest).

Extract the complete ZIP, open the extracted folder, and run
`Pebble for Sian.exe`. Keep everything included in the extracted folder together.

Pebble stores progress locally in `%APPDATA%\PebbleForSian`. Installing a newer
version does not remove progress.

## Updates

Pebble checks this repository's `latest.json` file at most once per day. When a
newer chapter is available, her storybook provides a link to its download page.
No update is installed automatically.

This repository is the release channel. The source code is not distributed here.
